 삼항 연산자
	(조건식) ? (참이면 오는 부분) : (거짓이면 오는 부분)
	(조건식) ? (참일때의 값) : (거짓일 때의 값);
	삼항 연산자는 사용부분 통째로가 값이다. 
	경우의 수가 두 가지일 때 사용하는 것이 좋다. 

제어문
	위에서 아래로만 흐르던 코드의 흐름을 제어하기 위해 사용하는 문장 
	
 	조건문(선택제어문)
		if ~ else if ~ else
		switch ~ case ~ default		

	반복문(반복제어문)

조건문
	조건에 따라 경우의 수를 나누기
	흐름 나누기
	선택지를 주어줄 때 사용
	특정한 상태 물어볼때 사용(입력한게 3이니? 비밀번호가 "apple"이니?)
	
if문 
	if(조건식1) {
		조건식1이 참일때 실행
	} else if(조건식2) {
		조건식2가 참일때 실행
	}
	...
	else {
		위의 조건식들이 모두 거짓일 때 실행 
	}

switch문
	경우의 수가 변수의 값 등호 비교로 인해 나뉠때 사용
	변수에 담김 값에 따라 알맞은 case로 이동하고 문장을 실행한다.
	하지만 그 밑에 case가 더 있다면 그 안에 있는 문장들도 실행된다.
	따라서 각 케이스를 나누어 주려면 알맞게 break;를 사용해야 한다. 
	switch(변수) {
	case 값 1:
		변수의 값이 값1일때 실행
		break;
	case 값 2:
		변수의 값이 값2일때 실행
		break;
	...
	default:
		위의 값들이 아닐 때 실행
	}

기타 연산자
	break : 속해있는 switch문이나 반복문을 탈출한다. (if문은 탈출x)

누적 대입 연산자(복합 대입 연산자, 누적 연산자)
	+=, -=, *=, /=
	
	int data = 10;
	data = data+5; 

	data+=5; //data의 5만큼 더해서 넣어라! 

증감 연산자
	(data+=1) >> data++;
	(data-=1)  >> data--;
	
	전위형
		더한 다음에 data 이용
		++data;

	후위형
		data++;
