컴파일러 해석 방향
	위에서 아래로, 좌에서 우로 

서식문자(format 문자)
	%d : 정수
	%c : 문자
	%f : 실수
	%s : 문자열

제어문자
	\n : Enter
	\t : Tab
	\' : '
	\" : "
	\\ : \
	

출력 메소드
	printf("서식문자열", 값);
		서식문자열은 서식문자가 포함된 문자열 
		뒤에 올 값은 서식문자 갯수만큼 뒤에 따라와야 한다.
		마지막에 줄바꿈을 하지 않는다. 
	print(값)
		괄호 안에 있는 값을 그대로 출력
		마지막에 줄바꿈을 하지 않는다.
	println(값)
		print()와 동일하나 마지막에 줄바꿈을 한다.

변수  
	값을 담는 저장공간 
	
	변수 		=		값;
(저장공간의	(대입연산자) 	(상수, 리터럴) 
이름)
	x		=		10;
 
=(대입 연산자)
	오른쪽에 있는 값을 왼쪽의 저장공간에 넣어라!

자료형
	저장공간의 종류, 타입 

	자료형 	type 	byte	예

	정수형	int		4	10,0,-274, ...
	실수형	float		4	10.274F, 0.0F, -71.117F 
			double 	8	10.274, 0.0, -71.117 
	문자형	char		2	'0', 'A', '가', ...

	문자열 	String 	??? 	"0", "Hello", ...

변수의 선언
	자료형 변수명 = 초기값;

변수 선언시 RAM
	int x = 10;
	x라는 이름의 저장공간이 RAM에 할당되고, 10이라는 값이 들어간다.
	공간이 만들어졌으므로 주소값이 부여되지만 알 수 없다. 

주소(C언어)
	저장공간(변수)이 RAM에 할당되면 고유한 값인 주소값이 부여된다. 

변수 선언시 주의사항
	같은 이름의 변수를 선언할 수 없다. 
	초기화를 해준다. 
	되도록 선언부(메소드 상단)에 한꺼번에 선언한다.

변수명 주의사항
	숫자는 맨 앞에 쓸 수 없다.
	특수문자는 쓸 수 없다. 
	공백을 쓸 수 없다.
	_(언더바)는 사용 가능하다.
	카멜 표기법 : noWhere

	되도록 소문자만 사용한다.
	되도록 한글은 사용하지 않는다. 
	뜻이 있는 명사형 단어를 선택한다. 
	(a,b,c,d...)X (num,height,data,...)O

	int data = 10;
	System.out.println(data); //10
