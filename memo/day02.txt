자바 프로그램의 구조
	프로젝트
		 패키지
			클래스(맨 앞글자가 대문자)
				메소드(이름 뒤에 소괄호())
					소스코드

{ }
	영역, 바디
	해당하는 클래스, 메소드 등의 영역을 나타낸다.
	*현재 선언중임을 나타낸다.

	public class 클래스명 {
		클래스의 영역	
	}

컴파일러 해석 방향
	위에서 아래로, 좌에서 우로 

서식문자(format 문자)
	%d : 정수
	%c : 문자
	%f : 실수
	%s : 문자열

제어문자
	\n : Enter
	\t : Tab
	\' : '
	\" : "
	\\ : \
	

출력 메소드
	printf("서식문자열", 값);
		서식문자열은 서식문자가 포함된 문자열 
		뒤에 올 값은 서식문자 갯수만큼 뒤에 따라와야 한다.
		마지막에 줄바꿈을 하지 않는다. 
	print(값)
		괄호 안에 있는 값을 그대로 출력
		마지막에 줄바꿈을 하지 않는다.
	println(값)
		print()와 동일하나 마지막에 줄바꿈을 한다.
